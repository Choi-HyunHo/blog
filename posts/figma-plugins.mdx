## Figma Token

피그마의 작업물은 마지막에는 개발자들에게 핸드오프가 됩니다. <br/>
그래서 넘어갈 때 기본적인 항목들이 전부 코드화가 돼서 넘어가야 됩니다.

**Figma Token**을 사용하면 등록해놨던 모든 항목들이 깔끔하게 **json** 파일로 나오게 되고 <br/>
추후에는 디자이너가 Github나 GitLab 이런 곳에다 올릴 수 있습니다.

그리고 파일을 올렸다면 넘겨받은 개발자가 json 데이터만 가지고 와서 작업을 진행 할 수 있습니다.

**최종적으로 디자이너가 색상을 변경하고 다시 푸쉬를 했을 때 해당 색상이 바로 프로젝트에 반영이 될 수도 있습니다.**
