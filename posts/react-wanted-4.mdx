## 테스트코드

1. Unit Test

    - 함수, 컴포넌트가 독립적으로 잘 작동하는지 확인하는 것

2. Integration Test
    - 여러 함수들 또는 여러 컴포넌트들이 같이 잘 동작하는지 확인하는 것
3. E2E test
    - 사용자인것마냥 테스트 하는 것
